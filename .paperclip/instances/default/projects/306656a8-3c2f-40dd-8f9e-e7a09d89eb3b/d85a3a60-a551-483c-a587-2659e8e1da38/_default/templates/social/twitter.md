# Twitter / X — Post Template

**Limit:** 280 chars per tweet (links count as 23 chars regardless of length)
**Format:** Single tweet or thread (summary as reply if needed)

---

## Template — Single Tweet

```
{Headline} [{Source}]
{article_url}
#Space #Spazio #AstroNews #NASA
```

**Example:**

```
SpaceX Starship completes first fully successful orbital test flight [SpaceFlightNow]
https://spaceflight.example/article
#Space #Spazio #AstroNews #SpaceX
```

---

## Template — Thread (when summary is available)

**Tweet 1 (main):**
```
{Headline} [{Source}]
{article_url}
#Space #Spazio #AstroNews #NASA
```

**Tweet 2 (thread reply — summary):**
```
{Summary up to 270 chars}…
```

**Example thread:**

Tweet 1:
```
Artemis III lunar landing target date set for 2026, NASA confirms [NASA.gov]
https://nasa.example/artemis
#Space #Spazio #AstroNews #NASA #ESA
```

Tweet 2:
```
La missione porterà i primi astronauti sulla Luna dal 1972, inclusa la prima donna. 
Il sito di atterraggio al Polo Sud lunare è ancora in fase di selezione finale…
```

---

## Rules

- Keep Tweet 1 under 280 chars including the t.co-wrapped link (23 chars) and hashtags
- Hashtags go **after** the link, never embedded in the headline text
- No more than 4–5 hashtags on the main tweet (algorithm penalises hashtag spam)
- Thread replies don't need hashtags — they inherit reach from Tweet 1
- Use Italian or bilingual phrasing to target Italian-speaking space enthusiasts
