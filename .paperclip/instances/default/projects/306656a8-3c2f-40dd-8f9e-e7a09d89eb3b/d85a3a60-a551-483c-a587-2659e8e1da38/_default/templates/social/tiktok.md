# TikTok — Post Template

**Caption limit:** 2200 chars (but keep it short — TikTok viewers rarely read long captions)
**Hook:** First 100 chars must grab attention immediately
**Video:** Required — the script skips TikTok if no video_url is provided
**Format:** Short news clip or animated infographic (15–60 seconds recommended)

---

## Template

```
🚀 {Short punchy hook — max 100 chars, statement or provocative question}

{1–2 sentence blurb — 100–150 chars, expands on the hook}

#space #spazio #nasa #razzo #scienza #tiktokscienza #esplorazionespaziale #fyp #foryou
```

---

## Examples

**Breaking news style:**
```
🚀 SpaceX ha appena lanciato 22 satelliti Starlink con un razzo già usato 18 volte!

Un record di riutilizzo per Falcon 9. Il futuro dei lanci spaziali è già qui.

#space #spazio #spacex #razzo #starlink #scienza #tiktokscienza #fyp #foryou
```

**Discovery style:**
```
🌌 Gli scienziati hanno trovato acqua su un asteroide per la prima volta!

Questo potrebbe spiegare come l'acqua sia arrivata sulla Terra miliardi di anni fa.

#space #spazio #nasa #asteroide #acqua #universo #scienza #tiktokscienza #fyp
```

**Question hook style:**
```
🚀 Sai quanto ci vuole per arrivare su Marte con le tecnologie attuali?

Circa 7 mesi. La NASA sta lavorando per ridurlo a settimane con propulsione nucleare.

#space #spazio #nasa #marte #esplorazionespaziale #cosmologia #fyp #foryou
```

---

## Rules

- Hook must work even without reading the caption — the video carries the message
- Keep the blurb to 1–2 sentences max — TikTok is not a reading platform
- Always include `#fyp` and `#foryou` — they help with algorithm distribution
- Use 8–13 hashtags total
- Video recommendations:
  - Vertical format (9:16), 1080×1920px
  - 15–30 seconds for breaking news; up to 60s for explainers
  - Add subtitles — most users watch without sound
  - Start with movement or text on screen in the first 2 seconds
- Best posting times: 12:00, 19:00, 21:00 (Europe/Rome)
